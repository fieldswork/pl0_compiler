--- Instructions for running lex.c ---
To compile and run, execute the following commands in a UNIX environment:
gcc driver.c lex.c
./a.out <name_of_input_file.txt>

For example, if your input file's name is "input.txt", execute the following commands:
gcc driver.c lex.c
./a.out input.txt

Alternatively, use the Makefile:
To compile and run, execute the following commands in a UNIX environment:
make
./a.out <name_of_input_file.txt>
