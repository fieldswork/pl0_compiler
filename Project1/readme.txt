--- Instructions for running vm.c ---
To compile and run the VM, execute the following commands in a UNIX environment:
gcc vm.c
./a.out <name_of_input_file.txt>

For example, if your input file's name is "input.txt", execute the following commands:
gcc vm.c
./a.out input.txt
